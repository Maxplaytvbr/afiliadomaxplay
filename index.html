Como usar o Gerador Max

1. Busque um título - Pesquise por filmes ou séries

2. Selecione - Toque no resultado para carregar

3. Escolha o template - Selecione um estilo visual

4. Edite - Personalize textos, cores e elementos

5. Gere - Clique em "Gerar" para exportar

6. Salve - Toque e segure a imagem para salvar
